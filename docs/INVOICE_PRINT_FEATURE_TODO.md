## نکات فنی

### API Endpoints مرتبط:
- `GET /invoices/business/{business_id}/{invoice_id}/pdf` - دریافت PDF فاکتور
  - Query parameters: `template_id`, `paper_size`, `orientation`, `disposition`
- `GET /businesses/{business_id}/print-settings` - دریافت تنظیمات چاپ
- `PUT /businesses/{business_id}/print-settings` - به‌روزرسانی تنظیمات چاپ

### فایل‌های مرتبط:
- `hesabixUI/hesabix_ui/lib/pages/business/new_invoice_page.dart` - صفحه فاکتور جدید
- `hesabixUI/hesabix_ui/lib/services/invoice_service.dart` - سرویس فاکتور
- `hesabixUI/hesabix_ui/lib/services/business_api_service.dart` - سرویس کسب‌وکار
- `hesabixUI/hesabix_ui/lib/core/api_client.dart` - کلاینت API
- `hesabixAPI/adapters/api/v1/invoices.py` - API endpoint فاکتورها
- `hesabixAPI/app/services/business_service.py` - سرویس کسب‌وکار (backend)

### مدل‌های مرتبط:
- `BusinessPrintSettings` - تنظیمات چاپ کسب‌وکار
- `ReportTemplate` - قالب‌های گزارش
