========================================
پیش‌نیازهای اجرای برنامه Flutter Windows
========================================

برای اجرای hesabix_ui.exe روی کامپیوتر دیگر:

1. سیستم عامل:
   - Windows 10 (نسخه 1809 یا بالاتر) - حداقل
   - Windows 11 - توصیه می‌شود

2. Visual C++ Redistributable (ضروری):
   - دانلود: https://aka.ms/vs/17/release/vc_redist.x64.exe
   - این فایل را باید روی کامپیوتر مقصد نصب کنید

3. فایل‌های برنامه:
   - کل پوشه Release را کپی کنید
   - مسیر: build\windows\x64\runner\Release
   - شامل: hesabix_ui.exe + تمام DLL ها + پوشه data

========================================
روش نصب:
========================================

1. Visual C++ Redistributable را نصب کنید
2. کل پوشه Release را کپی کنید
3. روی hesabix_ui.exe دوبار کلیک کنید

========================================
مشکلات رایج:
========================================

خطا: "VCRUNTIME140.dll was not found"
راه‌حل: Visual C++ Redistributable را نصب کنید

خطا: "0xc000007b"
راه‌حل: Visual C++ Redistributable x64 را بازنصب کنید
