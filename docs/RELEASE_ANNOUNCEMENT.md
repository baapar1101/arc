# اطلاعیه انتشار نسخه آزمایشی حسابیکس آرک

🎉 **حسابیکس آرک - نسخه آزمایشی اولیه منتشر شد!**

با افتخار اعلام می‌کنیم که نسخه اولیه آزمایشی **حسابیکس آرک** آماده استفاده است. این سیستم حسابداری جامع با استفاده از جدیدترین تکنولوژی‌ها طراحی و پیاده‌سازی شده است.

## ✨ ویژگی‌های اصلی

### 🚀 معماری مدرن
- **Backend**: FastAPI با Python 3.11+ (سریع و قدرتمند)
- **Frontend**: Flutter (کراس‌پلتفرم - وب، موبایل و دسکتاپ)
- **Database**: MySQL 8.0+ با پشتیبانی کامل از UTF-8
- **امنیت**: احراز هویت JWT و رمزگذاری پیشرفته

#### 💼 مدیریت فاکتورها
- فاکتور فروش و خرید
- برگشت از فروش و خرید
- فاکتور مصرف مستقیم و ضایعات
- فاکتور تولید

- پشتیبانی از مالیات و تخفیف

#### 📦 مدیریت موجودی و انبار

- مدیریت مشتریان و تامین‌کنندگان
