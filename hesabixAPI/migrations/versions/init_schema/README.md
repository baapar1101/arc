# میگریشن اولیه اسکیما

این فولدر شامل میگریشن اولیه کامل پایگاه داده است که تمام جداول را ایجاد می‌کند.

## ساختار

هر جدول یا گروهی از جداول مرتبط در یک فایل جداگانه قرار گرفته است:

- `01_users.py` - جداول کاربران و احراز هویت
- `02_currencies.py` - جداول ارزها
- `03_businesses.py` - جداول کسب‌وکارها
- `04_persons.py` - جداول اشخاص
- `05_fiscal_years.py` - جداول سال مالی
- `06_accounts.py` - جداول حساب‌ها
- `07_categories.py` - جداول دسته‌بندی‌ها
- `08_products.py` - جداول محصولات و قیمت‌ها
- `09_documents.py` - جداول اسناد
- `10_taxes.py` - جداول مالیاتی
- `11_bank_accounts.py` - جداول حساب‌های بانکی، صندوق و تنخواه
- `12_checks.py` - جداول چک‌ها
- `13_warehouse_documents.py` - جداول اسناد انبار
- `14_product_bom.py` - جداول فرمول تولید (BOM)
- `15_file_storage.py` - جداول ذخیره‌سازی فایل
- `16_support.py` - جداول پشتیبانی
- `17_email_config.py` - جداول تنظیمات ایمیل
- `18_document_numbering.py` - جداول شماره‌گذاری اسناد

## استفاده

فایل `__init__.py` تمام این فایل‌ها را به ترتیب import و اجرا می‌کند.

برای اجرای میگریشن:
```bash
alembic upgrade head
```

برای برگشت میگریشن:
```bash
alembic downgrade -1
```

## مزایای این ساختار

1. **سازماندهی بهتر**: هر جدول در فایل جداگانه برای بررسی راحت‌تر
2. **نگهداری آسان‌تر**: تغییرات در یک جدول فقط یک فایل را تحت تأثیر قرار می‌دهد
3. **خوانایی بهتر**: فایل‌های کوچک‌تر راحت‌تر خوانده می‌شوند
4. **قابلیت بررسی**: می‌توان به راحتی ساختار هر جدول را بررسی کرد
